# Advanced Git Commands

This section covers **advanced and powerful Git commands** used in real-world development and team workflows.

These commands help you:

- Safely undo changes
- Temporarily save work
- Clean up commit history
- Inspect file-level differences

## What You Will Learn

- How to undo commits without breaking history
- How to temporarily store unfinished work
- How to reorganize commits
- How to inspect changes before committing

## Important Concepts

- **History Safety**
  Some commands preserve history (revert), while others rewrite it (rebase).

- **Temporary Storage**
  Git allows you to save unfinished work without committing it.

- **Inspection Before Action**
  Git provides tools to inspect differences before making changes permanent.

## Commands Covered

- `git stash`
- `git revert`
- `git rebase`
- `git diff`
- `git restore`

## Important Warning ⚠️

Some advanced commands modify Git history.
Always understand the command before using it in shared repositories.
